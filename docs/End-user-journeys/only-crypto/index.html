Crypto trading only

Does your service offer only buying and selling of crypto? If so, this end user guide is for you. This guide helps you understand a sample journey your end user will see.

ðŸš¨Note: Winter's APIs are still in beta and under active development. We encourage you to send feedback to feedback@usewinter.com

What your end user will see#

  1. User signs up for crypto trading (create a Winter user account with our Account API)

  2. User inputs KYC info with our KYC API

  3. Fund crypto trading account

    • Options:
      • Instantly transfer cash to end user's Winter account from a business account using our Transfer API
      • Directly deposit into a Winter cash account from consumers bank account with our Transfer API (takes 2-3 business days)

  4. Quote crypto prices using our Quote API

  5. Trade crypto with our Buy API and Sell API

  6. View assets and P&L with our Assets API

  7. View history of trades with our Orders API

  8. Withdraw USD to bank account with our Transfer API

  9. Full yearly ledger for taxes with our Ledger API

    • Note: Winter does not offer 1099-Bs
